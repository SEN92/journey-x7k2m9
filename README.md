# رحلة المفتّش — HSE Journey Game v4.0

 لعبة تعليمية تفاعلية في السلامة المهنية بأسلوب 2D point-and-click مستوحى من لعبة Machinarium.

## الميزات

- **شخصية مفتّش** كرتونية متحركة تتحرك بين المحطات
- **4 بيئات صناعية**: مصنع الغاز، مصفاة النفط، المنصة البحرية، المبنى الإداري
- **معدات حقيقية**: صمامات، خزانات، مبادلات حرارية، مرجل، مراوح، درامات
- **67 محطة تعليمية** موزعة على:
  - NEBOSH IDip (52 محطة)
  - BCSP/ASP (15 محطة)
- **~335 سؤال** (5 لكل محطة) مع شرح تفصيلي
- **نوعان من الأسئلة**: اختيارات متعددة + ملء الفراغات
- **دعم اللغتين**: العربية والإنجليزية مع تبديل فوري (🌐)
- **واجهة RTL/LTR** تلقائية حسب اللغة المحددة
- **اختيار أسئلة تكيّفي**: يُفضّلAreas الضعيفة تلقائياً
- **تصنيف صعوبة**: d:1 (سهل)، d:2 (متوسط)، d:3 (صعب)
- **نظام رتب** من متدرّب إلى استشاري
- **مؤثرات صوتية** مولّدة بـ Web Audio API
- **حفظ التقدم**: يحفظ اللغة والأداء والإنجازات في المتصفح

## التشغيل

افتح `index.html` مباشرة في أي متصفح (Chrome, Firefox, Edge, Safari).

## النشر على GitHub Pages

1. أنشئ مستودعاً على GitHub
2. ارفع الملفات
3. فعّل GitHub Pages من الإعدادات (Settings → Pages → Source: main branch)

## التحكم

- **النقر**: التحرك إلى الموقع
- **الأسهم / WASD**: التحرك
- **Escape**: إغلاق نافذة الأسئلة
- **🌐**: تبديل اللغة

## التقنيات

- HTML5 Canvas
- JavaScript (Vanilla)
- Web Audio API (مؤثرات صوتية)
- Google Fonts (Cairo, Rajdhani)
